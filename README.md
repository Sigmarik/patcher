# Binary Patcher
## Description & Purpose
Simple binary patcher for example crackme-s.
## Building

Compile the project (linux):

`...# make`

Clean the project (linux):

`...# make clean`

Run the program (linux):

`...# make run`

Remove build folders (linux):

`...# make rmbld`

Return project to its original state:

`...# make rm`
